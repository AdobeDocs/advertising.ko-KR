# (새 UI) 업로드된 일괄 시트 및 오류 파일 삭제

일괄 시트 파일, 랜딩 페이지 유효성 검사 오류 파일 및 기타 오류 파일을 수동으로 삭제할 수 있습니다. 파일은 업로드되거나 생성된 후 30일 후에 자동으로 삭제됩니다.

>[!TIP]
>
>일괄 시트를 게시했지만 오류가 발생한 경우 모든 오류가 해결될 때까지 오류 파일을 삭제하지 마십시오.

1. 주 메뉴에서 **[!UICONTROL Setup]** \> **[!UICONTROL Bulksheets]**&#x200B;을(를) 클릭합니다.

1. 삭제할 각 파일 옆에 있는 확인란을 선택합니다.

1. 일괄 작업 도구 모음에서 **[!UICONTROL Delete]**&#x200B;을(를) 클릭합니다.

1. 확인 메시지에서 **[!UICONTROL Confirm]**&#x200B;을(를) 클릭하여 삭제를 확인합니다.

   선택한 일괄 시트 중 게시하도록 예약된 일괄 시트가 있는 경우 확인하기 전에 알림을 받습니다. 예약된 일괄 시트를 삭제하면 보류 중인 게시물이 취소됩니다.
