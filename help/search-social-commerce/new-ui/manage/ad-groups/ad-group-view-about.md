# (새 UI) [!UICONTROL Ad Groups] 보기에 대한 정보

*Beta 기능*

[!UICONTROL Manage] > [!UICONTROL Ad Groups] 보기는 계정의 모든 광고 그룹을 나열합니다.

데이터 테이블 위에서, 지정된 날짜 범위에 대한 보기의 모든 광고 그룹에 대해 총 3개의 지표가 있는 성능 그래프를 열 수 있습니다.
