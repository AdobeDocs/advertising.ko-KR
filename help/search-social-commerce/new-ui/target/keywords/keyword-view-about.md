# (새 UI) [!UICONTROL Keywords] 보기에 대한 정보

*Beta 기능*

[!UICONTROL Manage] > [!UICONTROL Keywords] 보기에는 두 개의 탭이 있습니다.

* [!UICONTROL Keywords] 탭에는 선택한 광고주 계정에 대해 필터링된 보기의 모든 키워드가 나열됩니다.

* [!UICONTROL Negatives] 탭에는 선택한 광고주 계정에 대해 필터링된 보기의 모든 부정적인 키워드가 나열됩니다.

* 키워드에 [레이블 분류 할당](/help/search-social-commerce/new-ui/reports/label-classifications-manage.md)
