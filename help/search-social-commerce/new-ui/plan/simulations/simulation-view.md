# 시뮬레이션 세부 정보 보기

*Beta 기능*

화면에서 그래프 또는 표로 시뮬레이션 세부 정보를 볼 수 있습니다.

1. 메인 메뉴에서 **[!UICONTROL Plan]>[!UICONTROL Simulations]**&#x200B;을(를) 클릭합니다.

1. 시뮬레이션 이름을 클릭합니다.

   시뮬레이션이 그래프 보기에서 열립니다.

1. (선택 사항) 그래프 보기에서 다음 중 하나를 수행합니다.

   * 각 지출 대상 수준(단계)에 대해 예상 데이터가 표시되는 지표를 변경하려면 **[!UICONTROL Y-Axis Metric]** 메뉴에서 새 지표를 선택하십시오.

   * (주별 시뮬레이션만 해당) 새 **[!UICONTROL Target Mid Point]** 값을 입력하면 해당 값을 사용하여 예측된 데이터를 볼 수 있습니다.

   * (주간 시뮬레이션만 해당) **[!UICONTROL Constraints]** 메뉴에서 *[!UICONTROL With Constraints]*, *[!UICONTROL Without Constraints]* 또는 *[!UICONTROL With and Without]* 데이터를 볼 것인지 선택합니다.

1. (선택 사항) 테이블 보기를 열려면 **[!UICONTROL Table View]** 탭을 클릭합니다.

1. (선택 사항) 적용 가능한 포트폴리오에 대한 읽기 전용 설정을 보려면 오른쪽 상단의 **[!UICONTROL Portfolio Settings]**&#x200B;을(를) 클릭하십시오.
