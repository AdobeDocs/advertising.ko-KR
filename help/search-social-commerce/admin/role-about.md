# 역할 기본 정보

역할은 사용자 유형이 수행할 수 있는 권한을 결정합니다. 각 사용자는 역할에 할당되며 역할이 허용하는 모든 활동을 수행할 수 있습니다.

을(를) 가진 사용자만 [!DNL Adobe] 계정 관리자 및 관리자 역할은 사용자에게 역할을 할당할 수 있습니다.
