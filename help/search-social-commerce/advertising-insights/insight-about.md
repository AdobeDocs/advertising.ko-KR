# 정보 [!UICONTROL Advertising Insights]

[!UICONTROL Advertising Insights] 활성 캠페인이 포함된 최적화되고 활성 포트폴리오에 대해 시각적으로 실행 가능한 데이터를 제공합니다.

각 인사이트는 온디맨드로 생성되며, 출력은 다운로드할 수 있는 파일이나 온스크린 보고서입니다. 인사이트 파일의 생성된 인스턴스를 삭제할 수 있습니다.
